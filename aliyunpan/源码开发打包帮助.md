### 小白羊v3版本源码帮助

v3采用 ts + vue3 + vite + electron 模板开发

#### 1.下载源代码

```
https://github.com/gaozhangmin/aliyunpan.git
```

#### 2.打开代码目录，安装依赖

```cmd
yarn install
```

#### 3.开发调试运行

开发调试需要提前准备阿里云盘的client_id和client_secret，填写到src/config.ts中

```cmd
yarn run dev
```

执行命令后会调起electron窗口，配合vscode正常开发调试即可

#### 4.打包

```cmd
yarn run build
```
